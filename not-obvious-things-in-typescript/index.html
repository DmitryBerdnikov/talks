Потеря типизации в Object.keys

* также в Object.entries


						const info = { title: 'about', text: 'some' }
				

						const info = { title: 'about', text: 'some' }

						Object.keys(info).forEach((key) => {
							const value = info[key]
						});

Ошибка

No index signature with a parameter of type 'string' was found on type '{ title: string; text: string; }' 


						const info = { title: 'about', text: 'some' }

						Object.keys(info).forEach((key) => {
							const value = info[key]
						});

Решение:


						const info = { title: 'about', text: 'some' }

						type InfoKeys = keyof typeof info

						Object.keys(info).forEach((key) => {
						  const value = key as InfoKeys
						});
